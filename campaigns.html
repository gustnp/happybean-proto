📋 새 캠페인 추가 방법

  1. campaigns/_template.js 파일을 복사
  2. 파일명을 캠페인 ID로 변경 (예: campaigns/happynew2026.js)
  3. 파일 안의 id: "template" → id: "happynew2026" 으로 수정
  4. 브라우저에서 index.html?c=happynew2026 으로 열기
  5. 편집 모드로 내용 수정 후 JS 내보내기로 저장
