# 해피빈 캠페인 프로토타입

캠페인 커스텀 페이지의 **기획서 + 시안 + 제작 스펙**을 링크 하나로 제공합니다.

## 보는 법

배포 링크 접속 후 상단 토글로 모드를 선택하세요.

| 모드 | 대상 | 내용 |
|---|---|---|
| **시안** | 클라이언트, 모두 | 완성 페이지처럼 스크롤하며 구조·카피·흐름 확인 |
| **기획 문서** | 기획/클라이언트 | 캠페인 목표·KPI·일정 + 블록별 기획 의도 표시 |
| **제작 스펙** | 디자이너/개발자 | 블록별 필요 에셋 목록, 인터랙션·콩 지급 조건 정의 |

특정 캠페인 접속: `?c=캠페인아이디` (예: `?c=kookmincall110`)

인쇄·PDF 버튼으로 현재 모드 화면을 그대로 PDF로 저장할 수 있습니다.

## 수정하는 법

- 카피/구성/컬러 수정 → `campaigns/캠페인아이디.js` 파일만 수정 (코드 지식 불필요, 텍스트 교체)
- 새 캠페인 → `campaigns/_template.js` 복사 후 작성, `index.html`에 script 한 줄 추가
- 상세 절차는 `CLAUDE.md` 참고 (Claude Code로 자동화 가능)

## 참고

- 회색 점선 박스 = 이미지 들어갈 자리 (설명 텍스트가 곧 에셋 요구사항)
- 파란 테두리 답변 카드 = 퀴즈 정답 표시
- GNB·푸터·댓글·기부금·공유 영역은 해피빈 공통 컴포넌트 (캠페인별 텍스트만 교체)
- 이 레포는 **비공개**입니다. 파트너사 정보가 포함되므로 외부 공유는 배포 링크로만.
